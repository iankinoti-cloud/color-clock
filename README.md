# Color Clock

A live clock built with React and Vite that displays the current time and date.

## Features

- Displays the current time (HH:mm:ss) updated every second
- Displays the current date in a readable format (e.g. Monday, April 28th, 2026)
- Animations styled with CSS keyframes

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm (comes with Node.js)

### Installation

```bash
npm install
```

### Running the App

```bash
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

### Building for Production

```bash
npm run build
```

## Dependencies

- [React](https://react.dev/) — UI library
- [date-fns](https://date-fns.org/) — Date formatting
- [Vite](https://vitejs.dev/) — Build tool and dev server
